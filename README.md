## GPIO

- Servo Brush D3
- Relay brush D45
- Stepper 1 dir D16, pul D17
- Stepper 2 dir D14, pul D15
- Ultrasonic HCSR04 N1 (trig D4 , echo D5)
- Ultrasonic HCSR04 N2 (trig D6 , echo D7)
- Bluetooth HC06 RX D0 TX D1
- IMU MPU6050 SDA D20 SCL D21
